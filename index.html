Detoxify-Telugu

Hate Speech Detection in Telugu & Tenglish

Detoxify-Telugu is an open-source platform for detecting toxic content in Telugu, Tenglish, and English social media inputs. Powered by BERT-based transformer models and a Streamlit UI, it enables end-to-end toxicity classification with YouTube scraping, data annotation, and real-time predictions.

git clone https://github.com/PavanYellathakota/detoxify-telugu.git
cd detoxify-telugu

# Step 2: Set up virtual environment
python -m venv venv
source venv/bin/activate        # (Windows: venv\Scripts\activate)

# Step 3: Install dependencies
pip install -r requirements.txt

# Step 4: Launch app
streamlit run app.py
