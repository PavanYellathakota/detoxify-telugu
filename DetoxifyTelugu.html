Detoxify-Telugu

Combating Toxicity in Regional Social Media Spaces

Detoxify-Telugu is a pioneering initiative aimed at detecting and mitigating hate speech, abusive language, and toxic behavior across Telugu social media platforms. Leveraging advanced NLP techniques, this platform identifies explicit offensive content and categorizes it for better moderation and community safety.

1. Home Page

Overview of the platform and its goals. Introduces the creator and project roadmap.

2. Data Collection

Scrape comments from YouTube using multiple video URLs with a robust Selenium-based scraper powered by GeckoDriver (Firefox). Features include dynamic loading of comments with 'Show more' clicks, random scrolling to mimic human behavior, and support for up to 1000 comments per video. Save results as a CSV file, with an option to append to existing datasets via a Streamlit UI.

3. Data Annotation

Use keyword-based automation or manual review to label data with toxicity type.

4. Data Cleaning & Balancing

Remove duplicates, fix spelling, and balance across toxicity classes using built-in scripts.

5. Model Training

Select model type (Binary or Multi-Class), choose pre-trained BERT variant, adjust hyperparameters, and train.

6. Model Evaluation

View precision, recall, F1-score, and confusion matrix on validation set. Option to evaluate any saved model.

7. Real-Time Prediction

Enter text or upload CSV for batch predictions. Get live model classification results with confidence scores.
